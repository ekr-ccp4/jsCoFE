Structure Solution Scenarios

Structure Solution in jsCoFE includes the following major steps:

  1. Data import
  2. Reflection data processing and Space Group Definition
  3. Asymmetric Unit Definition
  4. Phasing
  5. Density Modification
  6. Model Building
  7. Refinement
  8. Ligand Building
  9. Analysis

where the order of steps is flexible to a degree. For example, data import may be conducted any number of times at any point of the project, although good practices suggest import of all available data, such as reflection datasets, sequences and molecular models, in the very beginning of the project; Asymmetric Unit Composition, as well as the Space Group, remain hypothesis until the structure is solved, and may be corrected at any point; Refinement is often used as part of Phasing and so on. It is therefore possible to give only a very general set of structure solution scenarios, with the aim to demonstrate basic ideas behind structure solution project development in jsCoFE.
