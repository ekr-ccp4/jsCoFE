Automated Molecular Replacement Scenario

There are 3 automated molecular replacement pipelines in CCP4: BALBES, MrBump and MoRDa. All of them require the absolutely minimum input: Structure Revision and Sequence data objects, and, if successful, produce a Structure Revision with automaically found molecular model optimally positioned in the ASU.

The simplest Auto-MR scenario lools like below:

Here, first job imports merged HKL datasets, sequence and coordinate files, which are not used in this scenario. Then, second job defines Asymmetric Unit using one of the merged reflection datasets and sequence. The Asymmetric Unit Definition task produces Structure Revision, which contains only HKL data set and composition of the Asymmetric Unit. The Revision, together with the Sequence object, is then used in 3rd job (Automated MR with BALBES), which finds and prepares Molecular Model(s) and molecular replaces them in the reflection data.
